# Telegram-бот для записи в парикмахерскую "Народная"
## Описание проекта
Бот для автоматизации записи клиентов.

## Как запустить бота
1. Установите Python 3.11+
2. Клонируйте репозиторий
3. Установите зависимости: `pip install -r requirements.txt`
4. Создайте файл `.env` и заполните по примеру из `.env.example`
5. Запустите бота: `python bot.py`

## Административные команды
Список команд для управления
Как добавить мастера (/add_master).
Как изменить расписание (/set_schedule).
Как посмотреть записи (/view_bookings).
